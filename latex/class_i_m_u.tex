Starts the \mbox{\hyperlink{class_i_m_u}{IMU}} to begin reading values. 

This function starts the \mbox{\hyperlink{class_i_m_u}{IMU}} to it can read from the \mbox{\hyperlink{class_i_m_u}{IMU}} and calculate the rotation vector from the raw data. We are using a BNO085 so it is able to merge the value from the accelerometer and gyro to give us values in the i, j, k and r vectors.

\doxysubsection{Detailed Description}
This file contains the funciton defitions for the PID Controller. This includes get functions for the PID constants and the setpoint and a run task that calculates a new PWM value for the motor. This value is caluclated from the different between the current angle and current setpoint. The run function also handles dithering and integral value saturation.
